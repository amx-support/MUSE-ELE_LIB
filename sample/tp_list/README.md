# MUSEでの複数タッチパネル一括制御の実装

### 機能
NetLinxのデバイス結合のようにタッチパネルの一括制御を行えるようにします。<br/>

### 使用方法

プログラムフォルダに **ele_libフォルダ** をコピーし、メインプログラムから **import** します。<br/>

##### オブジェクト生成

`tpls = tp_list.TPList([デバイスリスト])`

デバイスリストは context.devices.get()で生成したインスタンスをリストで指定

###### ボタンイベント処理登録<br/>
`button_watch(ポート番号,チャンネル番号,コールバック関数,インスタンスID)`

**tp.port[x].button[y].watch()**でボタンイベント登録を行う<br/>
インスタンスIDを指定すると個別処理になる<br/>
指定しない場合はリスト全体の処理になる<br/><br/>


###### レベルイベント処理登録<br/>
